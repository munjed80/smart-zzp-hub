========================================
QUICK START
========================================

1. Clone repository:
   git clone https://github.com/munjed80/smart-zzp-hub.git
   cd smart-zzp-hub

2. Create .env:
   DB_USER=zzp_user
   DB_PASSWORD=<generate with: openssl rand -base64 24>
   JWT_SECRET=<generate with: openssl rand -base64 32>

3. Deploy:
   docker-compose up -d

4. Verify:
   curl http://localhost:4000/api/health

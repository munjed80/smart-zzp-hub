## Design Patterns

### Common Elements Across All Pages

1. **Header**: Always shows "Smart ZZP Hub" or "Smart ZZP Hub - Bedrijf" with logout button
2. **Navigation**: Context-aware buttons for related features
3. **Cards**: Clean bordered boxes for summaries and forms
4. **Tables**: Consistent column-based layout for data display
5. **Buttons**: Clear action buttons with Dutch labels
6. **Status Badges**: Color-coded status indicators

### Color Coding (Referenced in Descriptions)

- **Green** (âœ“): Positive values (income, profit)
- **Red** (âœ—): Negative values (expenses, costs)
- **Blue**: Links and invoice numbers
- **Gray**: Empty states and placeholders

### Responsive Behavior

All pages are designed to be responsive:
- Tables scroll horizontally on mobile
- Forms stack vertically
- Cards adapt to screen width
- Buttons remain accessible on all screen sizes
